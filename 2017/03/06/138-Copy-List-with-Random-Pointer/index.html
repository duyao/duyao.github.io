138. Copy List with Random Pointer

https://leetcode.com/problems/clone-graph/?tab=Description
可以和复制图一样，弄个map放节点，一次复制next,一次复制random
与复制图相比，复制链表是可以做到o(1)空间的

做法是遍历3次，
1）复制原节点，使得新节点是原节点的next
2）添加新节点的random
3）在原节点中删去添加的节点
前两次新节点都不是直接相连的，而是通过原节点相连接的
