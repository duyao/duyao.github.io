330. Patching Array

思路

贪心
设置miss为缺少的数字，sum为当前所有选中数字的的最小和
挑选miss的规则

  • 若数组中x <= sum + 1则选中x
  • 若数组没有，就选中sum + 1为miss
    直到sum > n
