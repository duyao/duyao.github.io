89. Gray Code

方法一

思路

  • n = 2

00 -> 01 -> 11 -> 10

  • n = 3

000 -> 001 -> 011 -> 010 ->||||| 110 -> 111 -> 101 -> 100

因此可以看出来求n的格雷码与n-1有关
n的格雷码可以分为两部分
第1部分是第1位，新添加的0或者1
第2部分是后面的数字，可以看出来是n-1的格雷码的结果

而n的结果集是镜面对称的
前一部分是0开头加上n-1结果集
后一部分是1开头加上n-1结果集，
而两部分结果集的顺序相反
